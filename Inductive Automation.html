Inductive Automation

18. 01. 04
posted by: Damselfly

 

Damselfly Solutions is a Certified Integrator of Ignition software by Inductive Automation. Ignition is a industrial application server, used to create systems that cover the full spectrum between HMI, SCADA and MES. Its unique architecture enables rapid project development and deployment. Damselfly Solutions has developed several solutions using Ignition and the customers have realized significant gains in productivity, machine efficiencies and overall real time knowledge of processes and products.

Let us create the System You Want

Ignition has been used to create industrial automation software systems covering the full spectrum between HMI, SCADA, MES, and industrial middleware. Our unique web and database-centric architecture lets the software be used across dozens of industries worldwide.


Cost Effective, Access for Everyone

Data analysis is the key to improving efficiency and saving money. But analysis is only useful if the right people can see it. With Ignition, you get unlimited clients and unlimited data points, so you never have to choose only a select few who get the system on their desktop. You may have as many members of your company access the data as you need.


IT Department Approved

Our software has been met with accolades from IT departments across the globe. Ignition is based entirely on modern, cross-platform technology, such as web (HTTP+SSL), SQL databases, and Java.


Easy to Use

Ignition has a drag-and-drop designer and uses familiar design paradigms such as the concept of "tags" for both real-time control and status, as well as historian functionality. The server installation takes only a minute or two, and there is no client installation (they're web-launched!). Getting started using a new industrial software system has never been easier. Just install, browse your PLCs using the included OPC server, and drag your tags onto screens.


Industry Leading Support

Let's put it this way: it is not uncommon for our support techs to get offers from customers for a free steak dinner if they are ever in town. Our knowledgeable support staff is available by phone 24/7, and our vibrant online community support forum is always free.


Security

Ignition was built with security in mind from the ground up. Communication channels are encrypted using SSL technology (like banks use). SCADA projects use advanced role-based authentication, and can integrate seamlessly with corporate network security using Microsoft Active DirectoryTM.


Stability

Ignition is a mature industrial application server that has been running in plants worldwide for many years (Ignition used to be called FactorySQL and FactoryPMI). Our extensive automated QA system ensures that every version we release stands up to our rigorous quality standards. And if an issue sneaks past, you can be assured that we will do everything within our power to fix it as soon as possible.


Scalability

Ignition is based on modern software architecture and systems: Java, web servers, databases and OPC-UA. Combined with advanced features like load-balancing clustering and client re-targeting, its scalability is remarkable. From embedded store-and-forward data collection applications to SCADA and MES systems that span the globe via corporate WANs, you can be sure that Ignition will fit in your enterprise.


Customize for Your Needs

From buttons and LED displays to interactive charts and PDF reports, from simple data logging to downtime tracking and recipe management, if you're looking for it, it's probably in Ignition. As a system designer, Ignition gives you the confidence to say "Sure, it can do that." Our integrated scripting environment and modular architecture makes sure that the even the most custom requirements can be met.


Peace of Mind

Advanced features like load-balancing redundant server clustering and store-and-forward logging system means that your clients stay running and your logging doesn't skip a beat.


A System That Will Last

When you implement an industrial software system, we understand that you are making a sizable investment of time and energy. When you choose software by Inductive Automation, you can rest assured that your system will be compatible with future versions, and that we will not issue an update that renders your system obsolete.
